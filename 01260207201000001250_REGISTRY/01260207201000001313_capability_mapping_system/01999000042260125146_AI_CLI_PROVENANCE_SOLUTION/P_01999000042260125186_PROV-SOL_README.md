## Quick Start

### What This Solution Adds

1. **AI Provenance Signals** (Heuristic)
   - Track AI-assisted file activity (session count, tool usage)
   - Detect AI-heavy files needing human review
   - Measure AI vs human investment

2. **Supersession Intent Detection** (Evidence Channel)
   - Scan prompts for "replace", "deprecate", "migrate", "remove"
   - Flag files marked for supersession/deprecation
   - Evidence for obsolescence determination

3. **Session Lineage Tracking** (Provenance)
   - Map file evolution across AI sessions
   - Track which AI CLIs touched which files
   - Temporal analysis of AI activity

## Heuristic Rules Added

### RS_AI_PROVENANCE_SIGNALS_V1
- **R_AI_PROV_001**: AI_GENERATED_HIGH_ACTIVITY (5+ sessions, 3+ edits)
- **R_AI_PROV_002**: AI_RECENT_TOUCH (touched in last 7 days)
- **R_AI_PROV_003**: AI_AGENT_DIVERSITY_HIGH (2+ AI CLIs)

### RS_HUMAN_TOUCH_SIGNALS_V1
- **R_HUMAN_001**: HIGH_HUMAN_INVESTMENT (5+ human commits, ≤2 AI sessions)
- **R_HUMAN_002**: AI_HEAVY_LOW_HUMAN_REVIEW (5+ AI edits, ≤1 human commit)

### RS_SUPERSESSION_INTENT_V1
- **R_SUPER_001**: SUPERSESSION_INTENT_DETECTED (migration_intent = true)
- **R_SUPER_002**: DEPRECATION_INTENT_DETECTED (deprecation_intent = true)
- **R_SUPER_003**: REMOVAL_INTENT_DETECTED (removal_intent = true)

## Safety & Privacy

✅ **Hash-only prompt storage** - SHA256 hashes, never raw text
✅ **Repo-scoped paths** - Filters to `C:\Users\richg\ALL_AI\` only
✅ **Graceful degradation** - Works even if logs missing
✅ **Minimal metadata** - Timestamp, tool name, file path, session ID only
✅ **No secrets** - Never stores tokens, API keys, or full prompts

---

## Performance

- **SQLite index**: <1MB for 10K records
- **Ingestion**: Incremental (only new log entries)
- **Query**: O(log n) via indexed lookups
- **Cache**: Evidence registry caches results
