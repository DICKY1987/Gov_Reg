## Running Tests

### All Tests
```bash
pytest tests/ -v
```

### With Coverage
```bash
pytest tests/ -v --cov=. --cov-report=html
```

### Unit Tests Only
```bash
pytest tests/unit/ -v
```

### Integration Tests Only
```bash
pytest tests/integration/ -v -m integration
```

### Specific Test File
```bash
pytest tests/unit/test_evidence_schema.py -v
```

### Tests Requiring Real Logs
```bash
pytest -m requires_logs -v
```

## Test Categories (Markers)

- `@pytest.mark.unit` - Fast, isolated unit tests
- `@pytest.mark.integration` - Integration tests requiring external files
- `@pytest.mark.slow` - Slow-running tests
- `@pytest.mark.requires_logs` - Tests requiring real AI CLI logs

## Writing Tests

### Unit Test Example
```python
import pytest

class TestMyFeature:
    def test_basic_functionality(self):
        assert True

    def test_with_fixture(self, sample_doc_id):
        assert sample_doc_id == "DOC-TEST-PROVENANCE-001"
```

### Integration Test Example
```python
import pytest

@pytest.mark.integration
class TestWithExternalFiles:
    def test_read_registry(self, mock_doc_id_registry):
        # Test using mock registry
        assert mock_doc_id_registry.exists()
```

## Fixtures

- `temp_dir` - Temporary directory for test files
- `temp_sqlite_db` - Temporary SQLite database
- `sample_doc_id` - Sample document ID
- `sample_file_path` - Sample file path

## Coverage Requirements

- **Target**: 80%+ code coverage
- **Exclusions**: Test files, conftest.py, virtual environment

## Test Data Privacy

- **No real prompts** - All test data uses synthetic examples
- **No real paths** - Uses sanitized test paths
- **No secrets** - No API keys or credentials in fixtures
