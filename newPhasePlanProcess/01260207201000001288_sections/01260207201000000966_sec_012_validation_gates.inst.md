# SEC-012: Global Validation Gates Instructions

**Schema File:** `sections/sec_012_validation_gates.json`  
**Section ID:** `sec_012_validation_gates`  
**Required:** Yes  
**Updated:** 2026-01-28

---

## What To Do

Define global validation gates that apply across all phases (in addition to phase-specific validations).

---

## How To Do It

### Standard Global Gates

Define these 4 gates (minimum):

**1. GATE-STRUCTURE-001: Directory Structure**
```json
{
  "gate_id": "GATE-STRUCTURE-001",
  "name": "Directory Structure Validation",
  "command": "python scripts/validate_structure.py --schema schemas/dir_structure.json",
  "timeout_seconds": 10,
  "success_criteria": {
    "exit_code": 0,
    "expect_stdout_regex": ["/all_required_dirs_present: true/"]
  },
  "evidence_path": ".runs/global_gates/structure.log",
  "must_pass": true,
  "run_after_phase": null
}
```

**2. GATE-COVERAGE-001: Global Coverage**
```json
{
  "gate_id": "GATE-COVERAGE-001",
  "name": "Global Coverage Threshold",
  "command": "pytest --cov=src --cov-report=term | grep TOTAL",
  "timeout_seconds": 300,
  "success_criteria": {
    "exit_code": 0,
    "expect_stdout_regex": ["/TOTAL.*9[5-9]|100%/"]
  },
  "must_pass": true,
  "run_after_phase": "PH-IMPLEMENT"
}
```

**3. GATE-SECURITY-001: Security Scan**
```json
{
  "gate_id": "GATE-SECURITY-001",
  "name": "Security Scan",
  "command": "bandit -r src/ -f json -o .runs/global_gates/security.json -ll",
  "timeout_seconds": 120,
  "success_criteria": {
    "exit_code": 0,
    "file_content_assertions": [{
      "path": ".runs/global_gates/security.json",
      "json_path": "$.results",
      "expect_value": "[]"
    }]
  },
  "must_pass": true,
  "run_after_phase": "PH-HARDENING"
}
```

**4. GATE-DETERMINISM-001: Determinism Check**
```json
{
  "gate_id": "GATE-DETERMINISM-001",
  "name": "Determinism Verification",
  "command": "python scripts/test_determinism.py --runs 3 --compare-outputs",
  "timeout_seconds": 600,
  "success_criteria": {
    "exit_code": 0,
    "expect_stdout_regex": ["/determinism_score: (0\\.[9][5-9]|1\\.0)/"]
  },
  "must_pass": true,
  "run_after_phase": "PH-PACKAGE"
}
```

### Gate Execution Policy

```json
{
  "gate_execution_policy": {
    "run_all_after_each_phase": false,
    "run_specific_after_phases": true,
    "fail_fast": true,
    "parallel_execution": false
  }
}
```

---

## Validation Rules

1. ≥4 global gates defined
2. Each gate has: gate_id, command, timeout, success_criteria, evidence_path
3. All gate_ids follow pattern: GATE-{NAME}-{NNN}
4. All evidence paths under `.runs/global_gates/`
5. Gate execution policy specified

---

## AI Agent Notes

- Global gates supplement phase-specific validations
- Run at specific points (after certain phases complete)
- Fail fast prevents wasted work downstream
