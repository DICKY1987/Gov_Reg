## What To Do

Provide operational procedures for installation, execution, maintenance, rollback, and troubleshooting.

---

## How To Do It

### Step 1: Installation Runbook

**Prerequisites:**
```json
{
  "prerequisites": {
    "os": ["Windows", "Linux", "macOS"],
    "runtime": "Python >=3.11, <3.13",
    "tools": ["git >=2.40", "docker >=24 (optional)"]
  }
}
```

**Installation Steps:**
```json
{
  "steps": [
    {"step": "1. Clone repository", "command": "git clone {{REPO_URL}}"},
    {"step": "2. Create virtual environment", "command": "python -m venv .venv"},
    {"step": "3. Activate venv", "command": ".venv\\Scripts\\activate"},
    {"step": "4. Install dependencies", "command": "pip install -r requirements.txt"},
    {"step": "5. Verify", "command": "python scripts/verify_install.py", "expect_exit_code": 0}
  ]
}
```

**Troubleshooting:**
```json
{
  "troubleshooting": [
    {
      "issue": "pip install fails with dependency conflict",
      "solution": "pip install --upgrade pip && pip install -r requirements.txt --force-reinstall"
    }
  ]
}
```

### Step 2: Execution Runbook

**Development:**
```json
{
  "development": [
    {"step": "1. Generate plan", "command": "python scripts/generate_plan.py --input {{INPUT}} --output {{OUTPUT}}"},
    {"step": "2. Validate plan", "command": "python scripts/validate_plan.py {{PLAN_FILE}}", "expect_exit_code": 0},
    {"step": "3. Execute (dry-run)", "command": "python scripts/execute_plan.py {{PLAN_FILE}} --dry-run"},
    {"step": "4. Execute (real)", "command": "python scripts/execute_plan.py {{PLAN_FILE}} --execute"}
  ]
}
```

### Step 3: Maintenance Runbook

**Backup:**
```json
{
  "backup": [
    {"step": "Backup state DB", "command": "sqlite3 .state/pipeline.db '.backup .state/backups/pipeline_$(date +%Y%m%d).db'"},
    {"step": "Backup decision ledger", "command": "cp .state/decisions.jsonl .state/backups/"}
  ]
}
```

**Upgrade:**
```json
{
  "upgrade": [
    {"step": "1. Backup", "command": "python scripts/backup_all.py"},
    {"step": "2. Pull changes", "command": "git pull origin main"},
    {"step": "3. Upgrade deps", "command": "pip install -r requirements.txt --upgrade"},
    {"step": "4. Verify", "command": "python scripts/verify_upgrade.py", "expect_exit_code": 0}
  ]
}
```

### Step 4: Rollback Runbook

```json

}
```

## Validation Rules

1. Installation runbook with ≥5 steps
2. Execution runbook for development
3. Maintenance runbook with backup/restore/upgrade
4. Rollback runbook defined
5. Troubleshooting guide with ≥3 common issues
