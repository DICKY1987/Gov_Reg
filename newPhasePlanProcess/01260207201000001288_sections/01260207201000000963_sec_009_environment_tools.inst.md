# SEC-009: Environment + Tool Configuration Instructions

**Schema File:** `sections/sec_009_environment_tools.json`  
**Section ID:** `sec_009_environment_tools`  
**Required:** Yes  
**Updated:** 2026-01-28

---

## What To Do

Define complete environment requirements, dependencies, and tool configuration.

---

## How To Do It

### Step 1: Set Target Environment

```json
{
  "target_os": ["windows", "linux", "macos"],
  "default_shell": "powershell",
  "primary_language": "python"
}
```

### Step 2: Configure Language Settings

```json
{
  "language_config": {
    "python": {
      "version_constraint": ">=3.11,<3.13",
      "venv_path": ".venv",
      "package_manager": "pip"
    }
  }
}
```

### Step 3: List Required Services

```json
{
  "required_services": [{
    "name": "sqlite_db",
    "type": "file",
    "path": ".state/pipeline.db",
    "validation_command": "sqlite3 .state/pipeline.db 'SELECT 1;'",
    "expect_exit_code": 0
  }]
}
```

### Step 4: List Dependencies

For each dependency:
```json
{
  "name": "pytest",
  "version": ">=7.4",
  "purpose": "unit/integration testing",
  "dev_only": true
}
```

**Standard dependencies:**
- pytest, pytest-cov (testing)
- ruff (linting)
- mypy (type checking)
- pyyaml, jsonschema (parsing)
- click (CLI)
- requests (HTTP)
- psutil (metrics)
- rich (output formatting)

### Step 5: Define Tool Profiles

```json
{
  "tool_profiles": {
    "preferred_ids": ["pytest", "ruff", "mypy"],
    "tool_definitions": [{
      "tool_id": "pytest",
      "executable": "pytest",
      "version_constraint": ">=7.4",
      "purpose": "unit/integration testing"
    }]
  }
}
```

### Step 6: Set Resource Requirements

```json
{
  "resource_requirements": {
    "minimum": {"ram_gb": 8, "disk_gb": 10, "cpu_cores": 2},
    "recommended": {"ram_gb": 16, "disk_gb": 50, "cpu_cores": 4}
  }
}
```

---

## Validation Rules

1. primary_language must be specified
2. ≥5 dependencies listed
3. All dependencies have: name, version, purpose
4. Resource requirements defined
5. Tool profiles list ≥3 tools

---

## AI Agent Notes

- This section defines reproducible environment
- All dependencies must be installable via package manager
- Version constraints prevent "works on my machine" issues
