```json
{
  "section_id": "sec_XXX_name",

  "required": true/false,
  "instructions_file": "sections/sec_XXX_name.inst.md",
  "section_content": { ... }
}
```

Each instruction file contains:

## Usage for AI Agents

1. **Load JSON schema** to understand data structure
2. **Read instruction file** to understand how to fill the section
3. **Fill JSON with actual data** following instructions
4. **Validate** against rules in instruction file
5. **Link** to other sections as needed

## Benefits

✓ **Separation of concerns** - Schema vs instructions  
✓ **Easy to maintain** - Edit markdown instructions without touching JSON  
✓ **AI-friendly** - Structured (JSON) + natural language (Markdown)  
✓ **Bidirectional linking** - JSON → Instructions, Instructions → JSON  
✓ **Clear naming** - File names explicitly show linkage  
✓ **Modular** - Update one section without affecting others
