## Core Modules

### 1. canonical_hash.py

**Purpose:** Deterministic content hashing for PFMS and file validation

#### Functions:

**\hash_canonical_data(data: Dict[str, Any]) -> str\**
- Compute deterministic SHA256 hash of structured data
- Returns 64-character lowercase hex string
- Rules: Dict keys sorted, JSON with sorted keys, UTF-8 encoding

**\hash_file_content(filepath: Path) -> str\**
- Compute hash of file content for drift detection
- Returns SHA256 hex of file bytes

**Example:**
\\\python
from govreg_core import hash_canonical_data

data = {"key": "value", "num": 42}
hash_result = hash_canonical_data(data)  # Returns 64-char hex
\\\
