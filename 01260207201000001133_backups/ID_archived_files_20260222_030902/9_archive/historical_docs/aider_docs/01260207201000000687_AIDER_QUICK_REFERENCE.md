## Quick Commands

### Start Aider
```bash
cd <project>
aider                           # Uses global config
aider --help                    # Show all options
aider --model ollama_chat/qwen2.5-coder:7b  # Override model
```

### Inside Aider
```
/help                           # Show all commands
/add <file>                     # Add file to context
/drop <file>                    # Remove from context
/ls                             # List files in context
/clear                          # Clear chat history
/tokens                         # Show token usage
/undo                           # Undo last change
/diff                           # Show pending changes
/commit                         # Commit changes
/exit                           # Quit
```

### Cleanup
```powershell
# Preview cleanup
.\cleanup-aider.ps1 -WhatIf

# Execute cleanup
.\cleanup-aider.ps1
```

## What Gets Ignored (.aiderignore)

✅ **Always excluded from AI context:**
- Archives, backups, old files
- AI tool state (`.aider/`, `.claude/`, etc.)
- Build artifacts (`__pycache__/`, `dist/`)
- Dependencies (`node_modules/`, `venv/`, `pipx/`)
- Large logs (`.aider.llm.history`)
- User directories (`Downloads/`, `Desktop/`)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| **Aider not found** | `where.exe aider` → Should show pipx path |
| **Model not responding** | `ollama list` → Check deepseek-r1 exists |
| **Slow responses** | Timeout set to 600s (max allowed) |
| **History too large** | Archive `~/Documents/aider-config/history/*.md` |
| **Duplicates** | Run `.\cleanup-aider.ps1` |

## File Size Monitoring

```powershell
# Check history file sizes
Get-ChildItem ~/Documents/aider-config/history/ | Select-Object Name, Length

# Archive if > 10MB
cd ~/Documents/aider-config/history/
mkdir archives
mv .aider.chat.history.md archives/.aider.chat.history.$(Get-Date -Format "yyyyMMdd").md
New-Item .aider.chat.history.md -ItemType File
```

## Best Practices

1. ✅ **Keep context small** - Only add files you're actively editing
2. ✅ **Use .aiderignore** - Exclude large dependencies and logs
3. ✅ **Review diffs** - Use `/diff` before `/commit`
4. ✅ **Archive history** - Prevent history files from growing too large
5. ✅ **Clean cache** - Run cleanup script monthly
